# JavaFX Snake
Aplikacja prezentuje podstawowe działanie gry w węża. Co 5ty punkt wąż przyśpiesza. 
Język: Java

Sterowanie: 

* A - skręcanie w lewo
* S - skręcanie w dół
* D - skecanie w prawo
* W - skręcanie w górę

### Struktura folderów

* src/Assets - folder przechowywujący pliki graficzne aplikacji
* src/SnakeGameFX - folder przechowywujący pliki .java
